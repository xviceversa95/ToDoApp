Примеры запросов для теста приложения: 

GET 158.255.0.33:8080/api/tasks - получение списка задач

POST 158.255.0.33:8080/api/tasks  - добавление задачи
{
    "title": "Название таски", 
    "id": 0, 
    "done": true
}

PUT 158.255.0.33:8080/api/tasks/0 - меняем статус задачи
{
    "done": false
}

PUT 158.255.0.33:8080/api/tasks/0 - меняем названием задачи
{
    "title": "New name"
}

DELETE 158.255.0.33:8080/api/tasks/0 - удаляем задачу
